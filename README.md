## What is this?

Official repository for the [ParadeDB website](https://www.paradedb.com), hosted on Vercel. Looking for ParadeDB itself? Head to [paradedb/paradedb](https://github.com/paradedb/paradedb).

## Getting started

Install the dependencies. We recommend using pnpm. If you want to use `npm`, just replace `pnpm` with `npm`.

```bash
pnpm install
```

Then, start the development server:

```bash
pnpm run dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser to view.
